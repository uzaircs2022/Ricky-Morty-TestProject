# Ricky & Morty Test Project
 
## Local Development (Mac OS / Ubuntu)

 - Ruby version 2.7.5
 - Rails version 6.0.6

**Project Level Dependencies**

 - bundle install

**Setting up the Database**

 - rails db:migrate 

**Running the Rails Server**

 - rails server


# API Endpoints

### Return Episodes
Method: `POST`
url:  `localhost:3000/rickymorties`
